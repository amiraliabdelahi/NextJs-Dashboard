## Install dependencies:

```js
npm install
# or
bun install
```

## Set up environment variables (.env):

```js
DATABASE_URL="mysql://user:password@localhost:3306/dbname"
AUTH_SECRET="your-secret"
```

## Push Prisma schema to your database:

```js
npx prisma db push
```
